kali破解WiFi密码

本示例中

AP：小小志的手机热点

Station：小小志的物理机

kali：VMware虚拟机

==所有设备均与他人无关==

前期准备

  1. 安装好的Kali Linux
  2. 支持监听模式的USB网卡

查看网卡是否支持监听模式

在终端输入 airmon-ng

上面列出了支持监听模式的无线网卡、可以看到wlan0支持监听模式

开启监听模式

在终端输入 airmon-ng start wlan0(其中 wlan0 为支持监听模式无线网卡的名字)

运行完成后可以输入 ifconfig 查看是否开启

如果网卡名后面加了mon 就说明此时的网卡已进入监听模式

查看WiFi网络

在终端输入 airodump-ng wlan0mon

此时会列出周围WiFi的详细信息，包括信号强度、加密类型、频道等。记住要破解的WiFi的频道号和BSSID。按ctrl+c结束搜索。

  • BSSID显示目标网络的MAC地址
  • PWR显示网络的信号强度。数字越高, 信号越好
  • 信标是网络发送以广播其存在的帧
  • #Data, 显示数据包数或数据帧数
  • ＃/s 显示过去10秒钟我们收集的数据包数量
  • CH显示网络所在的频道
  • ENC显示网络使用的加密。可以是WEP, OPN, WPA, WPA2
  • CIPHER显示网络中使用的密码
  • AUTH显示网络上使用的身份验证
  • ESSID显示网络名称

抓取握手包

终端输入 airodump-ng -c CH –bssid BSSID -w 路径 wlan0mon 开始抓包

-c指定频道号

–bssid指定路由器bssid

-w指定抓取的数据包保存位置

数据包正抓取

如果抓不到数据，新建一个终端输入 aireplay-ng -0 0 -c station -a bssid 网卡名 (一般为wlan0mon)

(0 WiFi设备无限次数,-0 8则攻击8次。

攻击原理是：先让设备掉线，设备会再自动连接，并发这个自动连接过程会进行三次握手，会发送tcp包（里面包含加密的密码数据），我方伪装成WiFi热点去窃取该数据包。我方窃取后即可用字典穷举法暴力破解加密的WiFi密码,数据包里面的密码是哈希加密的，哈希加密只能正向)

出现WPA handshake时，表示抓包成功

两个终端都按ctrl+c停止，不然那边会一直断网的

cd 打开抓包的目录 ls列出来，就看到我们的数据包了
