计网实验四
小小志不憨 Lv2
  2022-04-08 15:19:51 2022-04-08 15:19  
  • 实验 
 
  • 实验 
 

一、实验目的

  1. 能够在 Windows 操作系统环境下配置 IIS 服务器，并对 IIS 进行基本配置，架构 Web 服务器和 FTP 服务器，解决配置和管理 IIS 服务器时遇到的问题。
  2. 能够使用 HTML 编写简单网页。
  3. 能够正确描述应用层协议 HTTP、FTP 工作机理。

二、实验类型及环境

  1. 实验类型：验证性+设计性实验。
  2. 实验环境：接入局域网的 PC 一台，Windows 操作系统。

三、实验内容

（一）构建WWW服务器

​ 在本地计算机上配置 WWW 服务，建立一台 Web 服务器。完成 Web 站点的基本设置，自己设计一个简单网页并进行测试。

​ （1）在 Windows 操作系统下安装 IIS（Internet Information Services）。

​ （2）练习 IIS 基本管理操作，进行 Web 服务器基础配置，包括Web 站点、主目录、文档 三个选项的设置，了解其相关的选项功能.

​ （3）自行设计编辑一个网页文件，可以使用记事本、写字板、WPS 等文本编辑软件编辑 网页文档，网页内容包括自己的“班级，学号，姓名”等信息，能够反映字体、字号变化，段 落等属性，提倡加入图片等多媒体对象，文件命名为 index.html。

​ （4）将编辑好的网页文档放在到你架构的 Web 服务器的发布主目录下，使用本地浏览器 测试 Web 服务器配置正确性以及网页设计效果。

​ 要求要有 Web 服务器配置相关截图和浏览器显示你所设计网页的结果截图。

（二）构建FTP服务器

​ 参照上述 Web 服务器建立和配置方法，在 IIS 服务器上构架 FTP 服务，使其能作为服务器 正常运行。在 FTP 服务器发布目录下存放若干文件，可供客户端下载。

​ 使用本地浏览器访问 FTP 服务器，测试下载和上传文件。

​ 要求要有 Web 服务器配置相关截图。

（三）网络测试服务器

​ 与相邻的同学互相作为客户机，访问对方的 WWW 服务器和 FTP 服务器，截屏显示结果， 并加以分析说明。

（四）网络通信协议数据报截获与分析

​ 在你测试 Web 服务器或 FTP 服务器时，使用“实验三”中的软件 Wireshark 捕获网络通信 数据，并分析网络层、运输层和应用的协议报文格式，说明关键字段含义。

四、实验要求

  1. 遵守学术诚信，个人独立完成，严禁抄袭、复制他人成果。

  2. 为便于区分个人工作，说明自己实验环境，截图要真实，自行设置参数体现个性，避 免与他人重复。

  3. 结合上面实验内容，撰写实验报告，报告内容包含：

    （1）专业、班级、学号、姓名

    （2）实验名称

    （3）实验目的

    （4）实验方案：要给出自己的设计方案，实验步骤和内容。

    （5）实验结果：实验取得的结果，对结果进行分析描述。

    （6）实验总结：对实验中的关键知识点、难点、遇到的问题及解决方案进行剖析，总结 自己的理解和技能的提升。

  4. 实验报告撰写认真规范，文档排版规范见《实验报告模版》（学习通平台资源中）。文 档命名规则为“学号+姓名”，如“201616920312 王飞飞”，以 PDF 文档格式提交。
